3.字符串函数

  • 并置运算符 (.) 用于把两个字符串值连接起来。

下面的实例演示了如何将两个字符串变量连接在一起：

实例

<?php 
 $txt1="Hello world!";   <br>
 $txt2="What a nice day!"; <br>
 echo $txt1 . " " . $txt2; <br>
 ?>   //运行结果：Hello world! What a nice day!
  • PHP strlen() 函数
    有时知道字符串值的长度是很有用的。
    strlen() 函数返回字符串的长度（字符数）。
    <?php 
echo strlen("Hello world!"); 
?>  运行结果：12
  • PHP strpos() 函数
  • strpos() 函数用于在字符串内查找一个字符或一段指定的文本。
    如果在字符串中找到匹配，该函数会返回第一个匹配的字符位置。如果未找到匹配，则返回 FALSE。
<?php 
echo strpos("Hello world!","world"); 
?>   输出：6
  • PHP strpos() 查找字符串首次出现的位置
  • PHP strrpos() 获取指定字符串在目标字符串中最后一次出现的位置
$url='C:\web\apache2\htdoc\cat.jpg';
$pos=strrpos($url, '\\');              //截取文件输出名称：cat.jpg
echo substr($url, $pos+1);       //截取文件所在路径，输出结果：C:\web\apache2\htdoc\

    5.类

  • 类的定义
    class 类名
{
  //成员属性
  //成员方法
}
    类名后的“{}”中是==类的成员==，在类中声明得==变量是成员属性==，在类中声明得==函数是成员方法==
  • 实例化类

    $对象名 = new 类名([参数1，参数2，...]);
  1. 对象->成员 “->”对象成员访问符
<?php
class Person    //创建类
{
    public $name = 'Tom';   //成员属性
    public $age ;
}
$p = new Person ();
echo $p->name;     //获取默认属性值，输出结果：Tom
$p->name = 'Lucy'    //修改默认属性值
echo $p->name;      //输出结果：Lucy
$p->age=15;       //为属性赋值
echo $p->age;     //输出:15
unset ($p->age);     //删除属性值
var_dump(isset($p->age));   //检测属性是否设置，输出：bool(false)
var_dump($property_exists('Person','age'))；  //检验属性是否定义，输出：bool(true)
